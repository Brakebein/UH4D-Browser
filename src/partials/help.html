Mouse gestures

There are multiple mouse gestures available:

  • Hold: Rotate view, Click: Select item, Double click: Select and focus item
  • Hold: Pan view
  • Scroll: Zoom view
  • Click: Select item and open context menu
  • Click: Select/deselect multiple items

Navigation bar

In addition to the mouse gestures, there is also the possibility to navigate with the left mouse button pressed.

  • Standard navigation mode, see "Mouse gestures"
  • Rotate/orbit view with left mouse button pressed
  • Pan view with left mouse button pressed
  • Zoom view with left mouse button pressed

Collection – Memorizing images

Images can be memorized. All memorized images can be found within the "Collection" tab. To add/remove an image to/from the collection, you can

  • Hit the star icon ( ) within the search results list entry
  • Right-click within the viewport on an image and choose "Add to collection"/"Remove from collection"
  • Press the responsible button within the detail view
